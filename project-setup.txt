# AI Scheduler Project Setup Guide

## System Requirements
- Python 3.8+
- Node.js 16+
- PostgreSQL database (or Supabase for cloud)

## Backend Setup (Python)
1. Install dependencies:
   pip install -r requirements.txt

2. Set up environment variables (.env):
   SECRET_KEY=your-secret-key-here
   DATABASE_URL=postgresql://username:password@localhost:5432/ai_scheduler
   OPENAI_API_KEY=your-openai-api-key-here
   ALGORITHM=HS256
   ACCESS_TOKEN_EXPIRE_MINUTES=30
   PROJECT_NAME="AI Scheduler API"
   VERSION="1.0.0"
   API_V1_STR="/api/v1"
   ENVIRONMENT=development

3. Run the backend:
   uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

## Frontend Setup (Node.js)
1. Install dependencies (if package.json exists):
   npm install

2. Or install manually using frontend-dependencies.txt

3. Run the frontend:
   npm run dev

## File Structure
- requirements.txt - Python backend dependencies
- frontend-dependencies.txt - Node.js frontend dependencies  
- project-setup.txt - This setup guide
